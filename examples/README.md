# Examples

This folder contains complete sample documentation sets.

Use examples when you want to see how several templates work together in a realistic project.

## Included Examples

- `minimal-project/` - smallest useful docs set.
- `open-source-library/` - library with contributing, security, issues, and pull requests.
- `web-app/` - web app docs with support and security.
- `cli-tool/` - command-line tool docs.
- `docs-site/` - documentation website docs.
