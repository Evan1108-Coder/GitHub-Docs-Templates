# Contributing Templates

`CONTRIBUTING.md` tells people how to help with your project.

## How To Use These Templates

Copy one file into your own repository and rename it to:

```text
CONTRIBUTING.md
```

For example:

```text
CONTRIBUTING_simple.md -> CONTRIBUTING.md
```

## Template Choices

- `CONTRIBUTING_simple.md` - small project with simple rules.
- `CONTRIBUTING_detailed.md` - larger project with setup, tests, and PR process.
- `CONTRIBUTING_open_source.md` - community open-source project.
- `CONTRIBUTING_docs.md` - documentation-focused project.
