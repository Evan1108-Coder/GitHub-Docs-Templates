## Reporting A Vulnerability

Please do not open a public issue for security vulnerabilities.

Report security issues by emailing:

```text
[SECURITY_EMAIL]
```

Include:

- A description of the issue.
- Steps to reproduce.
- Any affected versions.
- Suggested fixes if known.
