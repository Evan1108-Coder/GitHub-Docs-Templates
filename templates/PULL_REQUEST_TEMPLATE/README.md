# Pull Request Templates

A pull request template helps contributors explain their changes.

## Where These Go

For one template, copy it into your repository and rename it to:

```text
.github/PULL_REQUEST_TEMPLATE.md
```

For multiple templates, use:

```text
.github/PULL_REQUEST_TEMPLATE/
```

Example:

```text
PULL_REQUEST_simple.md -> .github/PULL_REQUEST_TEMPLATE.md
```
