# Issue Template Files

Issue templates help people create useful GitHub issues.

## Where These Go

Copy templates into this folder in your own repository:

```text
.github/ISSUE_TEMPLATE/
```

For example:

```text
ISSUE_bug_report.md -> .github/ISSUE_TEMPLATE/bug_report.md
ISSUE_feature_request.md -> .github/ISSUE_TEMPLATE/feature_request.md
ISSUE_config.yml -> .github/ISSUE_TEMPLATE/config.yml
```
