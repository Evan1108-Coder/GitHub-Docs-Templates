# Choosing Templates

Use this guide when you are not sure which files your repository needs.

## Personal Project

Use:

- `README_minimal.md`
- `LICENSE_MIT.txt` or another license you understand

Optional:

- `CHANGELOG_simple.md`

## Open-Source Library

Use:

- `README_library.md`
- `LICENSE_MIT.txt`, `LICENSE_Apache_2_0.txt`, or another suitable license
- `CONTRIBUTING_open_source.md`
- `CODE_OF_CONDUCT_basic.md`
- `SECURITY_private_report.md`
- `ISSUE_bug_report.md`
- `ISSUE_feature_request.md`
- `PULL_REQUEST_detailed.md`
- `CHANGELOG_keep_a_changelog.md`

## Web App

Use:

- `README_web_app.md`
- `CONTRIBUTING_simple.md`
- `SECURITY_web_app.md`
- `SUPPORT_simple.md`
- `RELEASE_notes.md`

## CLI Tool

Use:

- `README_cli_tool.md`
- `CONTRIBUTING_simple.md`
- `ISSUE_bug_report.md`
- `PULL_REQUEST_simple.md`
- `CHANGELOG_simple.md`

## Docs Site

Use:

- `README_docs_site.md`
- `CONTRIBUTING_docs.md`
- `ISSUE_documentation.md`
- `PULL_REQUEST_documentation.md`
