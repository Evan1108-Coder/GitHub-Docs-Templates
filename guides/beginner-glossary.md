# Beginner Glossary

## README

The main file that explains what a project is, how to use it, and where to start.

## License

The legal permission for other people to use, copy, modify, or share your project.

## Issue

A GitHub conversation used to report bugs, request features, ask questions, or plan work.

## Pull Request

A proposed change to a repository.

## Contributor

Someone who helps with the project.

## Maintainer

Someone who reviews changes and helps manage the project.

## Changelog

A file that lists important changes between versions.

## Security Policy

A file that tells people how to report security problems safely.
