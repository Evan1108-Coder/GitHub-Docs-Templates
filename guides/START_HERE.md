# Start Here

Welcome. This guide helps you choose the right GitHub docs for your project.

## The Smallest Useful Set

For most beginner projects, start with:

- `README.md` - explains what the project is.
- `LICENSE` - explains how others may use the project.
- `CONTRIBUTING.md` - explains how people can help.

For public or collaborative projects, also consider:

- `CODE_OF_CONDUCT.md`
- `SECURITY.md`
- `.github/ISSUE_TEMPLATE/`
- `.github/PULL_REQUEST_TEMPLATE.md`

## How To Use A Template

1. Open a folder in `templates/`.
2. Read that folder's `README.md`.
3. Copy the template that matches your project.
4. Rename it to the standard filename.
5. Replace every placeholder.
6. Delete sections that do not apply.

## Example

If you copy:

```text
templates/README/README_minimal.md
```

Rename it in your own project to:

```text
README.md
```
